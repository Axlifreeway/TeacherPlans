# TeacherPlans

Документация и быстрый старт — в `teacherfactory/`:

- [`QUICKSTART.md`](teacherfactory/QUICKSTART.md) — установка и запуск за 5 минут
- [`GROQ_README.md`](teacherfactory/GROQ_README.md) — подключение бесплатного облачного провайдера
- [`ARCHITECTURE.md`](teacherfactory/ARCHITECTURE.md) — устройство пайплайна (RAG, провайдеры, валидация)
