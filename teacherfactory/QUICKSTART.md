# 🚀 Быстрый старт TeacherFactory

## Настройка за 5 минут

### Вариант 1: Groq (рекомендуется для работы с методистом)

**Преимущества:** быстро, не нагружает ПК, бесплатно с лимитами

1. **Получи API ключ** (бесплатно):
   - Зайди на https://console.groq.com/keys
   - Зарегистрируйся
   - Создай новый ключ

2. **Настрой ключ** (один из способов):
   
   **Способ A: Через переменную окружения** (рекомендуется)
   ```bash
   # Linux/Mac
   export GROQ_API_KEY="gsk_..."
   
   # Windows (PowerShell)
   $env:GROQ_API_KEY="gsk_..."
   ```
   
   **Способ B: Через файл .env**
   ```bash
   cp .env.example .env
   # Открой .env и вставь свой ключ вместо gsk_...
   ```
   
   **Способ C: Через config.local.toml**
   ```toml
   # Создай файл config.local.toml
   [groq]
   api_key = "gsk_..."
   model = "llama-3.1-8b-instant"
   ```

3. **Запусти приложение**:
   ```bash
   streamlit run src/teacherfactory/app.py
   ```

4. **Проверь в интерфейсе**:
   - В боковой панели должен гореть ✅ рядом с Groq
   - Выбери Groq как активный провайдер

---

### Вариант 2: Ollama (локально, полностью приватно)

**Преимущества:** полная приватность, нет лимитов, работает без интернета

1. **Установи Ollama**:
   - Зайди на https://ollama.ai
   - Скачай и установи

2. **Загрузи модель**:
   ```bash
   ollama pull llama3.1:8b
   ```

3. **Запусти сервер**:
   ```bash
   ollama serve
   ```

4. **Запусти приложение** (в другом терминале):
   ```bash
   streamlit run src/teacherfactory/app.py
   ```

5. **Проверь в интерфейсе**:
   - В боковой панели должен гореть ✅ рядом с Ollama

---

## Предоставление доступа методисту

### Шаг 1: Настрой Groq
Следуй инструкции выше для варианта 1.

### Шаг 2: Открой доступ к приложению

**Вариант A: Cloudflare Tunnel (рекомендуется)**

1. Установи cloudflared:
   ```bash
   # Linux
   curl -L --output cloudflared.deb https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb
   sudo dpkg -i cloudflared.deb
   
   # Mac
   brew install cloudflared
   ```

2. Запусти туннель:
   ```bash
   cloudflared tunnel --url http://localhost:8501
   ```

3. Скопируй ссылку из вывода (вида `https://random-name.trycloudflare.com`)
   и отправь методисту.

**Вариант B: Ngrok (проще, но временная ссылка)**

1. Установи ngrok: https://ngrok.com/download

2. Запусти:
   ```bash
   ngrok http 8501
   ```

3. Скопируй ссылку и отправь методисту.

### Шаг 3: Методист работает
- Методист открывает ссылку в браузере
- Никакой установки не требуется
- Все запросы идут через твой ПК к Groq API

## Решение проблем

### ❌ "Ни один LLM провайдер не доступен"

**Для Groq:**
- Проверь, что API ключ правильно скопирован (без лишних пробелов)
- Попробуй переустановить ключ в .env или config.local.toml
- Проверь интернет-соединение

**Для Ollama:**
- Убедись, что `ollama serve` запущен
- Проверь: `curl http://localhost:11434/api/tags`
- Перезапусти Ollama: `ollama serve`

### ❌ "Ошибка аутентификации Groq"

- Ключ истёк? Создай новый на https://console.groq.com/keys
- Проверь, что ключ начинается с `gsk_`

### ❌ Медленная генерация

**Для Ollama:**
- Используй GPU: в config.local.toml добавь `num_gpu = -1`
- Или переключись на Groq

**Для Groq:**
- Проверь интернет
- Используй модель 8b вместо 70b для скорости
