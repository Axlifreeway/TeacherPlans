# Groq API интеграция для TeacherFactory

## Что такое Groq?

**Groq** — облачный сервис, предоставляющий бесплатный доступ к современным LLM:
- **Llama 3.1** (8B и 70B параметров)
- **Mixtral 8x7B**
- **Gemma 2** (9B)

### Преимущества Groq

| Параметр | Ollama (локально) | Groq (облако) |
|----------|-------------------|---------------|
| **Требования к железу** | Нужна мощная видеокарта или много RAM | Не требуется |
| **Скорость** | Зависит от железа | Очень высокая (~500 токенов/сек) |
| **Лимиты** | Безлимитно | ~30 запросов/мин (бесплатно) |
| **Интернет** | Не нужен | Требуется |
| **Конфиденциальность** | Данные локально | Данные уходят на сервер Groq |
| **Доступ для других** | Нужен туннель (ngrok/Cloudflare) | Достаточно дать ссылку на Streamlit |

### Бесплатные лимиты Groq

На момент написания (2024):
- **Llama 3.1 8B**: ~30 RPM (requests per minute)
- **Llama 3.1 70B**: ~15 RPM
- **Mixtral 8x7B**: ~15 RPM

Актуальные лимиты: https://console.groq.com/docs/rate-limits

Для методиста этого более чем достаточно:
- 1 технологическая карта = 1 запрос
- Даже при 30 запросах в минуту можно сгенерировать 1800 карт в час!

## Настройка

### Шаг 1: Получение API ключа

1. Зарегистрируйся на https://console.groq.com/
2. Перейди в раздел **Keys**
3. Нажми **Create API Key**
4. Скопируй ключ (начинается с `gsk_...`)

### Шаг 2: Установка зависимости

```bash
cd /workspace/teacherfactory
poetry add langchain-groq
```

### Шаг 3: Конфигурация

Создай файл `config.local.toml` (или отредактируй существующий):

```toml
[groq]
api_key = "gsk_..."  # Твой ключ
model = "llama-3.1-8b-instant"
```

### Шаг 4: Запуск приложения с Groq

```bash
poetry run streamlit run src/teacherfactory/app_with_groq.py
```

В интерфейсе появится переключатель **Ollama / Groq** в боковой панели.

## Как это работает для методиста

### Сценарий использования

1. **Ты запускаешь приложение у себя на ПК:**
   ```bash
   poetry run streamlit run src/teacherfactory/app_with_groq.py
   ```

2. **Открываешь доступ через туннель:**
   
   **Вариант A: Cloudflare Tunnel (рекомендуется)**
   ```bash
   # Установи cloudflared
   # Создай туннель:
   cloudflared tunnel --url http://localhost:8501
   ```
   Получишь постоянную ссылку вида: `https://your-tunnel.trycloudflare.com`

   **Вариант B: ngrok (проще, но временная ссылка)**
   ```bash
   ngrok http 8501
   ```
   Получишь ссылку вида: `https://abc123.ngrok.io`

3. **Методист открывает ссылку в браузере:**
   - Никакой установки ПО не нужно
   - Работает с корпоративного ПК
   - Выбирает в интерфейсе **Groq (облако)**
   - Генерирует карты уроков

### Почему это лучше Telegram?

| Критерий | Telegram | Streamlit + Groq |
|----------|----------|------------------|
| **Интерфейс** | Текстовые команды | Визуальный UI с формами |
| **Предпросмотр** | Нет | Есть (структура урока, компетенции) |
| **Скачивание** | Файлы в чате | Кнопка Download с правильным именем |
| **Пакетная генерация** | Неудобно | ZIP со всеми картами |
| **Чат с документами** | Ограничен | Полноценный интерфейс |
| **Валидация компетенций** | Нет | Подсветка непроверенных кодов |

## Примеры использования

### Генерация одной карты

```python
from groq_client import generate_with_groq
from generator import load_index, retrieve_context, _lesson_search_queries

params = {
    "discipline": "Компьютерные сети",
    "specialty": "09.02.07",
    "course_number": 2,
    "group_name": "ИС-21",
    "students_count": 25,
    "lesson_topic": "TCP/IP протоколы",
    "lesson_number": 5,
    "date": "01.10.2024",
    "teacher_name": "Иванов И.И.",
    "lesson_type": "комбинированный урок",
    "lesson_kind": "лекция",
    "duration": 90,
}

# Поиск контекста в документах
db, bm25_data = load_index()
queries = _lesson_search_queries(params["discipline"], params["specialty"])
context = retrieve_context(db, bm25_data, queries)

# Генерация через Groq
card = generate_with_groq(params, context)
print(card.lesson_topic)
```

### Чат с документами

```python
from groq_client import stream_chat_with_groq
from generator import load_index, retrieve_context

db, bm25_data = load_index()
question = "Какие компетенции формируются по специальности 09.02.07?"
context = retrieve_context(db, bm25_data, question, k=15)

for chunk in stream_chat_with_groq(question, context):
    print(chunk, end="", flush=True)
```

## Troubleshooting

### Ошибка "API key not configured"

Проверь, что в `config.local.toml` указан ключ:
```toml
[groq]
api_key = "gsk_..."
```

### Ошибка "Rate limit exceeded"

Превышен лимит запросов. Подожди минуту или переключись на другую модель.

### Ошибка импорта `langchain_groq`

Установи зависимость:
```bash
poetry add langchain-groq
```

### Медленная генерация

- Проверь скорость интернета
- Попробуй модель `llama-3.1-8b-instant` (быстрее 70B)

## Безопасность

⚠️ **Важно:** При использовании Groq данные отправляются на сервера Groq AI.

- **Не используй** для конфиденциальных документов
- **Используй Ollama** для работы с чувствительными данными
- Groq соответствует стандартам безопасности, но данные покидают вашу инфраструктуру
