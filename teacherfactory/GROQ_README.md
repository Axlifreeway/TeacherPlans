# Groq API в TeacherFactory

## Что такое Groq?

**Groq** — облачный сервис LLM с очень быстрым инференсом (LPU-чипы) и бесплатным тарифом. На бесплатном тарифе доступны:

- **Llama 3.3 70B Versatile** — рекомендуемая «большая» модель
- **Llama 3.1 8B Instant** — быстрая и дешёвая (рекомендуется по умолчанию)
- **Llama 3.2 1B / 3B Preview** — самые лёгкие модели
- **Gemma 2 9B Instruct** — модель от Google
- **DeepSeek-R1-Distill-Llama-70B** — рассуждающая модель (chain-of-thought)
- Whisper Large v3 (распознавание речи) и другие — в TeacherFactory не используются

> Список моделей Groq меняется (модели уходят в `deprecated`, появляются новые). Актуальный реестр: <https://console.groq.com/docs/models>.

### Ollama vs Groq

| Параметр | Ollama (локально) | Groq (облако) |
|----------|-------------------|---------------|
| Требования к железу | Нужны RAM/GPU | Не требуется |
| Скорость | Зависит от железа | Очень высокая (LPU) |
| Лимиты | Без лимитов | См. rate limits ниже |
| Интернет | Не нужен | Требуется |
| Приватность | Данные локально | Данные уходят в Groq |
| Доступ методисту | Нужен туннель (ngrok / Cloudflare) | Достаточно ссылки на Streamlit |

### Бесплатные лимиты (Free Tier)

Groq нормирует:
- **RPM** — запросов в минуту;
- **RPD** — запросов в сутки;
- **TPM / TPD** — токенов в минуту/сутки.

Конкретные числа Groq меняет, и они зависят от модели — сверяйся с актуальной таблицей: <https://console.groq.com/docs/rate-limits>.

Для нашего сценария (1 технологическая карта = 1 запрос) даже минимальных лимитов хватает: можно генерировать карты пачками без боли.

## Настройка

### Шаг 1. Получить ключ

1. Зарегистрируйся на <https://console.groq.com/>.
2. Открой раздел **API Keys**.
3. Нажми **Create API Key**.
4. Скопируй ключ — он начинается с `gsk_…`.

### Шаг 2. Зависимости

`langchain-groq` уже прописан в `pyproject.toml`, дополнительно ставить ничего не нужно. Если установка проекта была без него:

```bash
poetry add langchain-groq
```

### Шаг 3. Прописать ключ

Любой из вариантов:

**A. Переменная окружения**

```powershell
$env:GROQ_API_KEY = "gsk_..."   # PowerShell
```

```bash
export GROQ_API_KEY="gsk_..."    # bash / zsh
```

**B. Файл `.env`** (в корне `teacherfactory/`, лежит в `.gitignore`):

```
GROQ_API_KEY=gsk_...
```

**C. `config.local.toml`** (рядом с `config.default.toml`, тоже в `.gitignore`):

```toml
[groq]
api_key = "gsk_..."
model = "llama-3.1-8b-instant"
```

> **Важно про секции TOML.** `api_key` и `model` для Groq обязаны лежать ровно под заголовком `[groq]`. Если положишь их под `[model]` — приложение Groq просто «не увидит» и молча уйдёт на Ollama.

### Шаг 4. Запуск

```bash
poetry run streamlit run src/teacherfactory/app.py
```

В сайдбаре появится переключатель «Ollama / Groq». Если ключ прописан верно — Groq будет с зелёной галочкой и доступен для выбора.

## Сценарий «доступ для методиста»

1. Запусти приложение у себя:
   ```bash
   poetry run streamlit run src/teacherfactory/app.py
   ```
2. Прокинь HTTP-туннель:

   **Cloudflare Tunnel** (рекомендуется — стабильнее и без аккаунта для quick tunnel):
   ```bash
   cloudflared tunnel --url http://localhost:8501
   ```

   **ngrok** (проще, но временная ссылка):
   ```bash
   ngrok http 8501
   ```
3. Отправь методисту полученную ссылку. Он работает в браузере, ничего ставить не нужно. Выбор провайдера (Ollama/Groq) делается в боковой панели.

### Почему Streamlit + Groq лучше Telegram-бота

| Критерий | Telegram | Streamlit + Groq |
|----------|----------|------------------|
| Интерфейс | Текстовые команды | Полноценный UI с формами |
| Предпросмотр | Нет | Структура урока, компетенции |
| Скачивание | Файл в чат | Кнопка с правильным именем |
| Пакетная генерация | Неудобно | ZIP со всеми картами |
| Чат с документами | Ограничен | Стрим + подсветка |
| Валидация компетенций | Нет | Подсветка непроверенных кодов |

## Рекомендации по выбору модели

| Модель Groq | Когда брать |
|-------------|-------------|
| `llama-3.1-8b-instant` | Дефолт. Быстро, дёшево, для типовых карт. |
| `llama-3.3-70b-versatile` | Сложные/спорные темы, когда 8B плывёт. |
| `gemma2-9b-it` | Альтернатива 8B, иногда лучше держит формат. |
| `deepseek-r1-distill-llama-70b` | Когда нужно «рассуждение» по сложной педагогической задаче. Медленнее. |

Сменить модель — поправь `model` в секции `[groq]` `config.local.toml`.

## Troubleshooting

**`Ни один LLM-провайдер не доступен`**
- Groq: убедись, что `api_key` лежит в секции `[groq]` (не `[model]`), и что в нём нет лишних пробелов / кавычек.
- Ollama: запущен ли `ollama serve`, есть ли модель из `config.default.toml` (по умолчанию `llama3.1:8b`).

**`Rate limit exceeded`** — кончилась квота на минуту/сутки. Подожди или переключись на другую модель / на Ollama.

**Модель `xxx` is deprecated/decommissioned** — Groq иногда выпиливает модели (так было с `mixtral-8x7b-32768`, `gemma-7b-it`). Замени на актуальную из <https://console.groq.com/docs/models>.

**Медленная генерация** — проверь скорость интернета. Свич с 70B на 8B обычно даёт прирост в разы.

## Безопасность

⚠️ При работе через Groq данные (промпт + контекст из документов) уходят на их серверы.

- Если документы содержат **персональные данные** студентов — используй **только Ollama**.
- API-ключ нигде не хардкодь в коде. `.env` и `config.local.toml` уже в `.gitignore`, не убирай их оттуда.
- Если ключ когда-либо попал в публичное место (commit, скриншот, чат) — **ротация** в <https://console.groq.com/keys> занимает 30 секунд.

## Альтернативные бесплатные API

Если Groq не подходит (лимиты, deprecated-модели, цензура контента) — на 2025–2026 в этой нише чаще всего смотрят на:

| Сервис | Что даёт | Особенности |
|--------|----------|-------------|
| **Google AI Studio** (Gemini) | Gemini 2.x Flash/Pro | Большой free-tier на запросы и контекст, нужен Google-аккаунт |
| **OpenRouter** | Маршрутизация к разным провайдерам, есть бесплатные модели | Один ключ на всё, удобно для экспериментов |
| **Together AI / Fireworks / DeepInfra** | Llama / Mixtral / Qwen | Дешёвый платный тариф (центы за миллион токенов) |
| **Hugging Face Inference Providers** | Разные модели через единый API | Есть бесплатные кредиты |

Добавить нового провайдера в проект — это новый класс-наследник `LLMProvider` в `llm_provider.py` + ветка в `get_llm_provider()`. Архитектура для этого готова.

## Итог

Groq покрывает основной use-case «дай методисту удобный доступ снаружи, не покупая железо»:
- Бесплатно (в рамках лимитов).
- Быстро (LPU).
- Не требует мощного железа у пользователя.
- Сохраняется Streamlit-интерфейс.

Минусы:
- Нужен интернет.
- Данные уходят на серверы Groq.
- Лимиты и список моделей меняются.

Максимальная приватность — **Ollama + Cloudflare Tunnel**. Удобство и скорость — **Groq**.
